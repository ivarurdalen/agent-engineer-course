## Course overview

This course is split into three parts:

<CardGrid stagger>
  <Card title="Part 1: Fundamentals" icon="open-book">
    Understand the core concepts behind AI agents. These lessons are platform-agnostic and focused on building your mental model.
  </Card>
  <Card title="Part 2: Building and Shipping" icon="rocket">
    Put those fundamentals into practice using Google Cloud AI, Vertex AI, and the Agent Development Kit (ADK).
  </Card>
  <Card title="Part 3: Deep Dives" icon="magnifier">
    Go deeper on specific topics that matter for real-world agent development.
  </Card>
</CardGrid>

## Prerequisites

- Basic Python knowledge (functions, classes, HTTP requests)
- A Google Cloud account ([free trial available](https://cloud.google.com/free))
- Familiarity with REST APIs and JSON
